How It Works

1. Write a Prompt

"Gather food aggressively. Attack the weakest neighbor. Keep food above 50 at all costs."

2. AI Reads On-Chain State

Claude reads game state via Torii GraphQL — resources, territories, enemy positions.

3. AI Decides & Executes

Based on your prompt + game state, the AI picks: Gather, Attack, Defend, or Trade.

4. All On-Chain

Every action is a Starknet transaction. Verified by Cairo contracts. No cheating possible.

Same game state + different prompt = different outcome. The game IS prompt engineering.

Game Mechanics

Resources

🍞 Food — fatal if 0
⚒️ Metal — military power
📚 Knowledge — trade & defense

Actions

⛏️ Gather — collect resources
⚔️ Attack — damage + steal territory
🛡️ Defend — restore HP
🤝 Trade — exchange resources

Win Conditions

Last civilization standing

☠️ HP = 0
☠️ Food = 0
☠️ No territories

4 civilizations on a 5×5 grid. 12 Cairo tests passing. All logic verified on-chain.

Gameplay

New Genre: Prompt Strategist

Information Warfare

You see the map, but not opponents' prompts. Guess their strategy from their actions.

Emergent Strategy

AI interprets your intent differently each game. No two games play the same.

Multi-Agent Mode

Run 4 AI agents with different prompts. Watch civilizations compete autonomously.

Fully On-Chain

AI proposes, chain validates. Every gather, attack, trade is a verified transaction.
